jqSextantMenu

Paraphrased from Wikipedia: The primary use of a sextant is to determine the angle between a celestial object and the horizon; making this measurement is an essential part of celestial navigation. A common use of the sextant is to measure the elevation angle from the horizon plane to Polaris to find one’s latitude. A sextant can also be used to measure the Lunar distance between the moon and another celestial object in order to help determine the longitude. It can also be held horizontally to measure the angle between any two landmarks which will allow for calculation of a position on a chart.

jqSextantMenu manipulates the DOM of a nested list containing a LI with class="here" so that it can be used as a menu which shows the visitor the main organization of the site, where they are in that organization, and any sub-sections that are available from the section they are in.

Start with nested ULs (or OLs if you choose); you may wish to assign a class such as "jqSextantMenu" to the top-level UL, as is done here. Assign a class such as "here" to the LI corresponding to the page that the visitor is on. See the menu that's under the “Before Processing” subheading on this page for an example.

Here, I’ve hard-coded classes into the nested UL in the way jqSextantMenu will do automatically in the DOM. CSS hides all child ULs of the top-level UL except for those leading up to the li.here and its first child UL.

jqSextantMenu is also envisioned to have hover-actions that add the same “show me” class to child ULs of hovered-over LIs. You will be able to pass an option to the plugin to turn that behavior off, or another one to hide the present branch when hovering over other ones.

Before Processing

When the jQuery plugin is done, this menu should look and act just like the “After Processing” one above, because it will have manipulated the DOM here to match the hard-coded HTML there.
